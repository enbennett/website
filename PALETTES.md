# Per-Page Color Palettes

Derived from the reference photos in `Webpage Theme Ideas.pdf`. Each page is its
own palette; the site is unified by shared typography + spacing, not color
(see `REDESIGN_PLAN.md` §2). Hex values are starting points — easy to nudge once
we see them on real layouts.

Each palette lists color **roles** so they map straight onto CSS custom
properties (e.g. `--bg`, `--accent`).

---

## Home — Record Player  *(warm walnut + teal vinyl glow)*
From the wooden turntable console with the glowing teal record.

| Role | Hex | Notes |
|------|-----|-------|

| `--wood-light` | `#A9714B` | highlighted wood grain |
| `--accent` (mustard gold) | `#D8A93A` | record label, knob rings, links |
| `--accent-bright` | `#EFC659` | label glow / hover highlights |
| `--chrome` | `#D9D2C4` | knobs / tonearm metal |
| `--ink` (text) | `#F4ECDD` | warm cream text |
